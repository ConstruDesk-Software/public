# POLÍTICA DE PRIVACIDADE – CONSTRUDESK

## 1. DADOS COLETADOS

O ConstruDesk coleta apenas dados necessários para funcionamento do aplicativo, incluindo:

- Dados de cadastro (nome, e-mail, identificadores);
- Dados de obras, funcionários, tarefas e registros;
- Informações técnicas do dispositivo;
- Logs de uso e diagnóstico.

## 2. FINALIDADE DO USO DOS DADOS

Os dados são utilizados para:

- Operação do aplicativo;
- Armazenamento e sincronização de informações;
- Suporte técnico;
- Cumprimento de obrigações legais.

## 3. COMPARTILHAMENTO DE DADOS

Os dados podem ser compartilhados apenas com:

- Provedores de infraestrutura (servidores, banco de dados);
- Plataformas de pagamento, quando necessário;
- Autoridades legais, mediante ordem judicial.

O ConstruDesk não vende dados pessoais.

## 4. SEGURANÇA DA INFORMAÇÃO

São adotadas medidas técnicas para proteção dos dados.  
O usuário reconhece que nenhum sistema é completamente seguro.

## 5. RETENÇÃO E EXCLUSÃO

Os dados são armazenados enquanto a conta estiver ativa.

O usuário pode solicitar exclusão da conta, ciente da perda definitiva dos dados.

## 6. DIREITOS DO TITULAR (LGPD)

O usuário pode solicitar:

- Acesso aos dados;
- Correção de informações;
- Exclusão, quando aplicável.

## 7. ALTERAÇÕES

Esta Política pode ser atualizada periodicamente.

## 8. CONTATO

Dúvidas podem ser enviadas pelos canais oficiais do aplicativo.
